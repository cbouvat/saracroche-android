Saracroche est une application Android qui vous protège des appels indésirables en bloquant automatiquement les appels de spam. Elle est conçue pour être simple, efficace et respectueuse de la vie privée.

<b>Fonctionnalités :</b>
• Bloque automatiquement les numéros de spam
• Application Android native avec une interface Material Design 3 moderne
• Respectueuse de la vie privée : aucune donnée d'appel n'est collectée ou transmise
• Mises à jour régulières de la base de données des numéros de spam
• Utilise l'API Call Screening d'Android pour des performances optimales

<b>Confidentialité et sécurité :</b>
Saracroche donne la priorité à votre vie privée. L'application fonctionne localement sur votre appareil et ne collecte aucune information personnelle ou donnée d'appel. Toute la détection de spam est effectuée sur l'appareil en utilisant une base de données régulièrement mise à jour.

<b>Comment ça marche :</b>
L'application utilise le service de filtrage d'appels d'Android pour identifier et bloquer automatiquement les numéros de spam connus avant qu'ils puissent vous déranger. Lorsqu'un appel arrive, Saracroche vérifie le numéro dans sa base de données de spam et le bloque s'il est identifié comme spam.

<b>Exigences :</b>
• Android 10 (niveau API 29) ou supérieur
• Permission d'être définie comme application Téléphone par défaut pour le filtrage d'appels

<b>Open Source :</b>
Saracroche est un logiciel open source. Vous pouvez examiner le code, contribuer ou le compiler vous-même à partir du code source disponible sur GitHub.
